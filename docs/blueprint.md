# **App Name**: MooviTech

## Core Features:

- Product Catalog: Display a catalog of products with filtering and sorting options (marca, memoria o precio)
- Product Details Page: Show detailed information for each product, including images, price, description, and specifications.
- Shopping Cart: Allow users to add items to a shopping cart and view a summary of their selected products. Even if is not fully implementes for this stage
- Contact Form: Provide a contact form for users to submit inquiries and receive support.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey sophistication and modernity.
- Background color: Dark gray (#37474F) for a soft, uncluttered background.
- Accent color: Light purple (#F3E5F5) for text and interactive elements to ensure contrast.
- Headline font: 'Poppins' (sans-serif) for a precise, contemporary feel, suitable for headlines.
- Body font: 'PT Sans' (sans-serif) for body text, selected for its modern look and readability in longer text blocks. (pairing of 'Poppins' and 'PT Sans')
- Use clean, minimalist icons to represent different product categories and UI actions.
- Maintain a clean, minimal layout with plenty of white space to highlight products and reduce visual clutter.
